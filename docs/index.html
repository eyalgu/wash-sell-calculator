Input

CSV format & input guide

Columns: date, action, source, shares, pricePerShare, transactionType, acquiredDate, notes (optional)

FieldValues
actionBUY, SELL
sourceShareworks, Computershare, Other
transactionTypeRSU_VEST, SELL_TO_COVER, IPO_SALE, OPEN_MARKET_SALE, ESPP_PURCHASE, ESPP_SALE

How to enter common transactions:

RSU vest + sell-to-cover — enter as two rows on the same date: a BUY for the full vest, then a SELL for the shares sold to cover taxes. For BUY rows, set acquiredDate = date.

2025-09-01,BUY,Shareworks,100,100.00,RSU_VEST,2025-09-01
2025-09-01,SELL,Shareworks,40,99.50,SELL_TO_COVER,2025-09-01

Open market sale — set acquiredDate to the original vest/purchase date of the lot being sold.

2025-11-15,SELL,Shareworks,60,70.00,OPEN_MARKET_SALE,2025-09-01

Selling multiple lots at once — split into one row per lot (matches 1099-B reporting). Each row has the lot's own acquiredDate and share count.

2025-11-15,SELL,Shareworks,60,70.00,OPEN_MARKET_SALE,2025-09-01
2025-11-15,SELL,Shareworks,60,70.00,OPEN_MARKET_SALE,2025-10-01
2025-11-15,SELL,Shareworks,60,70.00,OPEN_MARKET_SALE,2025-11-01
